# Mar de Névoa

O **Mar de Névoa** é a faixa marítima entre as [Cidades-Estado Arcanas](../10 - Kaelthar - Nações/Cidades-Estado Arcanas.md), coberta por uma névoa mágica constante. A névoa não é meteorológica — é resíduo de séculos de atividade arcana acumulada na região, e ela torna a navegação sem guia arcano praticamente impossível.

Por consequência, o Mar de Névoa funciona como um perímetro defensivo natural não-intencional: poucas frotas hostis cruzam-no sem perderem rumo, navios ou tripulação. Para magos das cidades, é uma zona de testes; para forasteiros, é um obstáculo.

## O Que Define a Região

- Faixa marítima entre as cinco Cidades-Estado Arcanas
- Coberta por névoa mágica permanente
- Defesa natural contra invasões marítimas
- Acessível apenas com guia arcano licenciado por uma das cidades
