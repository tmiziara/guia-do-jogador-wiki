# Culto do Fogo Original

O **Culto do Fogo Original** é a tradição espiritual mais antiga dos [Domínios dos Fumegantes](../10 - Kaelthar - Nações/Domínios dos Fumegantes.md). Sustenta que os dragões foram os primeiros seres conscientes do planeta, anteriores a qualquer outra raça, e que o calor vulcânico é a forma "natural" do mundo.

Para os fiéis mais radicais, a Estrela e o calor que ela mantém são uma imposição artificial — algo construído contra a vontade original do planeta. Por essa lógica, o avanço do gelo nas eras recentes não é catástrofe: é correção. Trata-se de uma posição minoritária mesmo entre Dragonborn, mas presente o suficiente para causar atrito com religiões solares de Valdren.

## O Que Define a Organização

- Tradição espiritual antiga entre Dragonborn dos Fumegantes
- Sustenta que dragões antecedem todas as outras raças conscientes
- Vê o frio crescente como retorno natural, não como ameaça
- Em desacordo direto com a [Igreja de Vaelun](../07 - Valdren - Organizações/Igreja de Vaelun.md)

## Como é Visto

- **Por outros Dragonborn:** tradição respeitável, ainda que polêmica.
- **Por sacerdotes solares:** heresia perigosa, especialmente quando apoiada por dragões antigos.
