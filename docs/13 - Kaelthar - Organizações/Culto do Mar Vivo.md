# Culto do Mar Vivo

O **Culto do Mar Vivo** não é exatamente uma religião organizada — é uma tradição compartilhada por capitães e marinheiros das [Mil Ilhas](../10 - Kaelthar - Nações/Mil Ilhas.md) e dos portos de [Kaelthar](../02 - Continentes e Grandes Regiões/Kaelthar.md). Para os fiéis, o oceano é uma entidade consciente que pode ser propiciada ou provocada, e nenhum gesto antes ou depois de uma viagem é desperdiçado.

Os rituais são simples e cotidianos: oferendas no embarque, libações no retorno, luto formalizado para os "que foram para o Mar". Em portos maiores, o navio-templo de **Thessala das Ondas** ancora periodicamente para receber fiéis e bênçãos.

## O Que Define a Organização

- Culto popular dos marinheiros de Kaelthar
- Oceano tratado como entidade consciente
- Coexiste com religiões oficiais sem conflito direto
- Mosteiro flutuante de Thessala das Ondas viaja entre portos

## Como é Visto

- **Por marinheiros:** prática inegociável; viajar sem oferenda atrai azar.
- **Por sacerdotes de cultos formais:** superstição inofensiva, ocasionalmente útil para conter pânico no mar.
